Nextflow 시작하기

설치

Java 8 이상이 필요합니다.

curl -s https://get.nextflow.io | bash

첫 번째 워크플로우

간단한 Hello World 예제:

#!/usr/bin/env nextflow

process sayHello {
    input:
    val x

    output:
    stdout

    script:
    """
    echo 'Hello $x!'
    """
}

workflow {
    Channel.of('World', 'Korea') | sayHello | view
}
